# Food Delivery App

A modern and feature-rich Food Delivery mobile application built with Flutter and Firebase, designed to provide a seamless ordering experience across Android and iOS platforms.

## Features

### Authentication
- Email & Password Authentication
- Google Sign-In
- Persistent User Sessions

### Food Ordering
- Browse food categories
- Real-time search functionality
- Product details screen
- Add to cart and quantity management

### Cart & Checkout
- Promo code support
- Delivery time estimation
- Multiple payment methods
- Order summary and confirmation

### Orders
- Real-time order tracking
- Animated order status updates
- Order history

### User Experience
- Favorites management
- Loyalty rewards system
- Reviews and ratings
- Dark & Light Mode
- English & French localization
- Connectivity monitoring

## Tech Stack

- Flutter
- Dart
- Firebase Authentication
- Provider
- SharedPreferences
- Google Fonts
- Flutter Animate

## Architecture

The application follows an MVC architecture and uses Provider for state management.

### Providers

- AuthProvider
- CartProvider
- OrdersProvider
- FavoritesProvider
- LoyaltyProvider
- ReviewsProvider
- PromoProvider
- ThemeProvider
- ConnectivityProvider
- LocaleProvider

## Installation

Clone the repository:

```bash
git clone https://github.com/mohammedbouaouin-1/Food-Delivery-App-Mobile.git
```

Navigate to the project:

```bash
cd Food-Delivery-App-Mobile
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```

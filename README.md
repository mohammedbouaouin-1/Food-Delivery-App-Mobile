# Food Delivery App

A complete Food Delivery mobile application built with Flutter and Firebase.

## Features

- Firebase Authentication
- Google Sign-In
- Cart Management
- Order Tracking
- Loyalty Rewards
- Favorites System
- Reviews & Ratings
- Dark/Light Theme
- English & French Localization

## Tech Stack

- Flutter
- Dart
- Firebase Authentication
- Provider
- SharedPreferences

## Getting Started

```bash
flutter pub get
flutter run
```
